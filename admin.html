📖 사용 가이드

  1. "파트너 관리 시작" 클릭 → 18개 카드 리스트가 보입니다.
  2. 각 카드 우측의 "편집" 으로 업체명·로고·요약·설명·링크 수정.
  3. "저장" 클릭 시 자동으로 GitHub에 반영되어 1~2분 후 라이브 사이트 업데이트.
  4. 로고는 편집 모달의 "📤 업로드" 버튼으로 GitHub에 바로 업로드 가능.
  5. 우상단 "🏷 카테고리 관리" 에서 카테고리 추가/수정/삭제.
  6. 일괄 변경 후 "🚀 GitHub에 반영" 으로 한 번에 반영 가능.
🔀 카드 정렬:
편집 모달의 "저장" 클릭 시 GitHub에 자동 반영됩니다.
순서 변경·삭제·정렬 모드 변경은 우상단 "🚀 GitHub에 반영" 필요.

GitHub 설정 (이미지 업로드용)

💡 Personal Access Token 발급 방법:
1. github.com/settings/personal-access-tokens 접속
2. "Generate new token" → Fine-grained 선택
3. Repository access: 해당 레포 하나만 선택
4. Permissions → Contents: Read & write
5. 생성된 토큰(ghp_... 또는 github_pat_...) 복사
브라우저 localStorage에 저장됩니다. 어드민 로그인을 통한 1차 보호만 적용되니 fine-grained PAT 사용을 권장합니다.
