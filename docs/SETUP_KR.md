# 견적서 발송 자동화 설정 가이드

## 1) 기본 개념
- **원본 견적서**: 질문에 첨부한 Google Spreadsheet를 템플릿으로 사용합니다.
- **관리자 시트**: 견적 발행 요청과 이력관리를 담당합니다.
- 발행 시 템플릿을 복제하여 고객사 전용 URL/PDF URL을 만듭니다.

## 2) 관리자 시트 준비
1. 새 Google Spreadsheet 생성 (예: `견적서 Admin`).
2. 시트 `견적요청` 생성 후 헤더 입력:
   - A: 고객사명
   - B: 담당자
   - C: 메모
   - D: 토큰
   - E: 웹URL
   - F: PDF URL
   - G: 발행일시
3. Apps Script 편집기에서 `apps-script/quotation_system.gs` 내용을 붙여넣습니다.

## 3) 설정값 확인
`CONFIG`에서 아래 값을 운영 환경에 맞게 수정하세요.
- `TEMPLATE_SPREADSHEET_ID`: 원본 견적서 ID
- `TARGET_GID`: 고객에게 보여줄 시트 gid
- `OUTPUT_FOLDER_ID`: 견적서 사본 저장 폴더 ID (선택)

## 4) 실행
- 문서를 다시 열면 상단 메뉴에 `견적서 자동화`가 생깁니다.
- `선택 행 견적서 발행`: 현재 선택한 행 1건 발행
- `요청 시트 전체 일괄 발행`: 토큰이 비어 있는 행 모두 발행

## 5) 고객 전달 방식
- **웹 URL**: E열의 링크 전달
- **PDF URL**: F열 링크 열어서 PDF 다운로드 후 전달

## 6) 이력관리
- 자동으로 `견적이력` 시트가 생성/누적됩니다.
- 저장 항목: 발행일시, 고객사명, 담당자, 메모, 토큰, 스프레드시트ID, 웹URL, PDF URL, 생성자

## 7) 권장 운영 정책
- 고객사 수정/재발행은 **새 토큰으로 재발행**하여 이력 보존
- 링크 권한(ANYONE_WITH_LINK)을 사내 보안정책에 맞춰 제한 검토
- 월 1회 이력 CSV 백업
