# 홈페이지 제작 견적서 발송/이력관리 자동화

이 저장소는 **구글 스프레드시트를 견적서 원본으로 사용**하면서,
- 고객 발송용 URL 생성
- PDF 생성
- 견적 이력(일시/고객사/개별 URL) 관리
를 자동화하기 위한 초기 구현을 포함합니다.

## 구성

- `apps-script/quotation_system.gs` : Google Apps Script 코드
- `docs/SETUP_KR.md` : 한국어 설치/운영 가이드

## 기대 흐름

1. 원본 견적 스프레드시트(서식/숫자 포함)를 준비
2. 관리자 시트에서 고객사명 입력
3. 스크립트 실행으로 고객사별 견적서 사본 생성
4. 고객 전달용 웹 URL / PDF URL 생성
5. 이력 시트에 발행일시/고객사/토큰/URL 자동 기록
