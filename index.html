카카오 예약 시작 가이드

카카오톡에서 환자가 예약하면 플래너에 자동으로 나타납니다.
아래 순서대로 한 번만 설정하면 완료됩니다.

채널 연결
→
파트너 등록
→
스토어 등록
→
상품 등록
→
항목·가격 등록
설정 단계
1
카카오 채널 연결
병원의 카카오 채널을 검색해서 연결합니다.
플래너 설정 → 파트너사 관리 → 카카오 탭 이동
파트너 연결 버튼 클릭
사업자번호 입력 후 조회 → 채널 선택 → 연결하기
2
파트너 등록
채널 선택 후 병원 정보를 확인하고 등록합니다.
병원명·대표자명·사업자번호·전화번호·주소 자동 입력 확인
등록하기 버튼 클릭 → 왼쪽 파트너 목록에 카드 생성됨
3
스토어 등록
병원 위치를 지도에서 검색해 스토어를 생성합니다.
파트너 카드 → 이어서 관리하기 클릭
장소 검색으로 병원 위치 선택 → 등록하기
4
상품 등록
카카오 예약으로 받을 진료·시술 스케줄을 선택합니다.
목록에서 원하는 스케줄 선택 → 등록하기
플래너에 등록된 스케줄 중 파트너 노출이 켜진 것만 목록에 표시됩니다.
5
항목(item) 등록
상품 아래에 세부 진료·시술 항목을 추가합니다.
등록할 항목 선택 (복수 선택 가능)
선택한 항목 등록 버튼 클릭
6
가격(price) 등록
항목별 가격 이름과 설명을 입력하면 설정이 완료됩니다.
여기까지 완료되면 카카오톡에서 환자 예약을 받을 수 있습니다!
예약 처리 방법
확인
예약 확인
플래너 캘린더에서 카카오 아이콘이 붙은 예약을 찾아 클릭하세요. 환자 정보와 확정/거절 버튼이 나타납니다.
확정
예약 확정
확정 버튼 → 확인 팝업에서 확정 클릭. 환자 카카오톡으로 예약 확정 알림이 자동 발송됩니다.
거절
예약 거절
거절 버튼 → 사유 선택 (예약 마감 / 장소 운휴 / 구매자 요청 / 단순 변심 / 직접 입력) → 확인.
입력한 거절 사유는 환자에게 카카오톡으로 전달됩니다. 정중하게 작성해 주세요.
예약 상태 흐름
예약 신청 → 확정 → 완료 정상 흐름
예약 신청 → 거절 병원에서 거절
확정 → 취소 확정 후 취소
자주 묻는 질문
파트너 → 스토어 → 상품 → 항목 → 가격까지 모두 등록되어 있는지 확인하세요. 하나라도 빠지면 예약을 받을 수 없습니다.
이미 확정·거절·취소된 예약입니다. 예약 신청 상태에서만 확정/거절이 가능합니다.
네, 입력한 거절 사유는 환자의 카카오톡으로 전달됩니다. 정중하게 작성해 주세요.
플래너에 자동으로 반영됩니다. 별도 조치가 필요 없습니다.
가능합니다. 여러 카카오 채널을 각각 연결할 수 있으며, 왼쪽 파트너 목록에서 관리하세요.
